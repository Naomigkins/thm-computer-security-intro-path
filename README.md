# thm-computer-security-intro-path
Beginner-friendly cybersecurity learning path covering networking, OS basics, offensive/defensive security tools, and career foundations.

# 🔰 Introduction to Computer Security — Learning Path

This beginner-friendly path aims to provide a solid introduction to multiple foundational areas in Computer Security. It is structured to give a clear understanding of the essential concepts, tools, and career options in cybersecurity.

---

## 📘 What This Path Covers

### 🔹 1. Computer Networking & Cryptography
- Basics of network communication  
- Protocols, routing, and addressing  
- Encryption, hashing, keys, and core crypto principles  

### 🔹 2. Operating Systems Fundamentals
- **Windows** basics  
- **Active Directory** concepts  
- **Linux** command-line essentials  

### 🔹 3. Offensive Security Tools & System Exploitation
- Reconnaissance  
- Enumeration  
- Vulnerability identification  
- Introduction to ethical hacking tools  

### 🔹 4. Defensive Security Solutions & Tools
- SIEM basics  
- Threat monitoring  
- Host/network defense strategies  

### 🔹 5. Cybersecurity Careers
- SOC Analyst  
- Penetration Tester  
- Security Engineer  
- Digital Forensics  
- Governance, Risk & Compliance (GRC)  

---

## 🎯 Purpose of This Repository
This repo will track progress, notes, tools, and hands-on exercises completed throughout the **Intro to Computer Security** learning path.
